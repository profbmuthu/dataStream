Day 1: Linux Lab:

Try to get used to Linux commands. Hadoop is built on Linux. ATTEMPT ANY FOUR.

  • Write a shell script that gets executed displays the message either “good morning” “good afternoon” “good evening” depend upon time at which user logs in.

  • Write a non recursive shell script which accept any number of argument and print them in the reverse order (ex:if the script is named rags then executing rags A B C should produce C B A on the standard output
